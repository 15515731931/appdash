Welcome

Apptrace is a performance and debug tracing suite for Go.

It allows for tracing the end-to-end performance of hierarchically structured applications (like distributed web apps). Apptrace enables you to, for example, see detailed information of each HTTP request and SQL query made by an entire distributed web application.

Getting Started

To get started, click on the Traces tab at the top of this page. When a trace is collected it will show up there.

Tip: Try right clicking on a span to bring up the context menu!

More

Apptrace is an open-source project hosted on GitHub, and you can use the issue tracker to file an issue or request new features.
